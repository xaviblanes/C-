# TAD matriu d'enters
En matemàtiques, una matriu és una taula rectangular de nombres o, més generalment, d'elements d'una estructura algebraica de forma d'anell. 
Un exemple de matriu de 2 files i 3 columnes:
~~~
int m[4][4] = {
  { 0,  1, 7, 0 },
  { 2, -2, 3, 0 },
  { 5, -5, 1, 7 },
  { 4,  0, 1, 9 }
};
~~~
